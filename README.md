\# Citi Technology Software Development Job Simulation



\## Overview

This project was completed as part of the \*\*Citi Group Technology Software Development Job Simulation\*\* on \[Forage](https://www.theforage.com/).



\## What I Built

\- A Java application that queries the \*\*Dow Jones Industrial Average (DJIA)\*\* stock price from Yahoo Finance every 5 seconds

\- Stock prices and timestamps are stored in a \*\*queue data structure\*\*

\- Built a \*\*JavaFX\*\* GUI to display the stock data

\- Used \*\*Gradle\*\* as the build system



\## Technologies Used

| Technology | Purpose |

|---|---|

| Java 17 | Main programming language |

| Gradle 9.5 | Build system |

| JavaFX 17 | GUI framework |

| Yahoo Finance API | Stock data |

| JUnit Jupiter | Testing |



\## How to Run

1\. Clone the repository:

git clone https://github.com/bhawanagurjar452/citi-forage-tech-simulation.git

2\. Navigate to the project folder:

cd citi-forage-tech-simulation

3\. Run the application:

gradle run --no-configuration-cache
